STEP 1: Install Java 21 (5 minutes)
────────────────────────────────────

1. Open this link in your browser:
   https://adoptium.net/temurin/releases/?version=21

2. Download: Windows x64 JDK .msi installer

3. Install it (check "Set JAVA_HOME" option)

4. Verify installation:
   Open NEW PowerShell window and run:
   
   .\INSTALL_JAVA21.ps1

   You should see:
   ✓ Found Java 21 at: C:\Program Files\...
   ✓ Java 21 is ready!


STEP 2: Build & Run Backend (3 minutes)
────────────────────────────────────────

Run this command:

   .\BUILD_AND_RUN.ps1

Or manually:

   mvn clean install -DskipTests
   mvn spring-boot:run

Backend will start on: http://localhost:8080/api


STEP 3: Start Frontend (2 minutes)
───────────────────────────────────

Open a NEW PowerShell window:

   cd frontend
   npm install
   npm run dev

Frontend will start on: http://localhost:5173

🗄️ DATABASE OPTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Option A: H2 (In-Memory - Default)
───────────────────────────────────
No setup needed! Just run:
   mvn spring-boot:run -Dspring-boot.run.profiles=dev

H2 Console: http://localhost:8080/api/h2-console
   JDBC URL: jdbc:h2:mem:cyberaudit_db
   Username: sa
   Password: (blank)


Option B: PostgreSQL (Production)
──────────────────────────────────
1. Install PostgreSQL
2. Create database:
   
   CREATE DATABASE cyberaudit_db;
   CREATE USER cyberaudit_user WITH PASSWORD 'your_password';
   GRANT ALL PRIVILEGES ON DATABASE cyberaudit_db TO cyberaudit_user;

3. Update src/main/resources/application.yml:
   
   spring:
     datasource:
       url: jdbc:postgresql://localhost:5432/cyberaudit_db
       username: cyberaudit_user
       password: your_password

4. Run:
   mvn spring-boot:run

Issue: "Port 8080 already in use"
Solution: netstat -ano | findstr :8080
         taskkill /PID <PID> /F

Issue: "mvn command not found"
Solution: Use mvn.cmd instead of mvn

Issue: "npm command not found"
Solution: Install Node.js from https://nodejs.org/
